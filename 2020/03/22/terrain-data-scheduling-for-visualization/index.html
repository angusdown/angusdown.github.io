terrain-data-scheduling-for-visualization

发表于 2020-03-22 分类于 algorithm

地形四叉树瓦片金字塔数据的调度编程实现上有两种方式，一是建立一个四叉树数据结构，保存瓦片上下层之间的拓扑关系，在一个节点中保存父节点和四个子节点的指针，另一种是用散列映射(hash_map)数据结构，其中瓦片的索引作为键，瓦片的智能指针作为值。四叉树数据结构在调入一个瓦片时必须先生成其父瓦片指针，在删除一个瓦片时必须先删除其所有子瓦片指针，不利于LRU算法的有效实施，而散列映射数据结构则要灵活得多。算法流程图如下所示。
